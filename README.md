# Dark Web Appsec Fingerprinting Demo

This repository contains supporting materials for the talk:

## "Secure by Obscurity Fails: Code Reuse & Vulnerability Clustering in Dark Web Apps"

Contents:
- Simulated marketplace HTML datasets
- DOM fingerprinting script
- Slide outline preview

This demo shows how template reuse across hidden-service applications
can be detected using simple fingerprinting techniques.

The dataset is synthetic and created for research demonstration purposes only.
